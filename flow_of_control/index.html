1. Introduction to Flow of Control

In programming, the Flow of Control (or Control Flow) refers to the exact order in which individual statements, instructions, or function calls are executed or evaluated by the Python interpreter.

1. Sequential

Top to Bottom. Line by Line. No skipping.

2. Conditional

Making choices. Skipping blocks of code based on Booleans.

3. Iterative

Looping. Executing the exact same block of code multiple times.

The Law of Indentation

How Python understands groupings.

2. Use of Indentation

Structure through Whitespace

Unlike languages such as Java or C++ that use curly brackets { } to define blocks of code, Python uses strict indentation (whitespace at the start of a line).

All statements that share the exact same indentation level are considered part of the same logic block.

if x > 10:
    print("Inside Block A")
    x = x - 1
print("Outside completely!")

The standard is 4 spaces per indentation level. Never mix tabs and spaces!

Sequential Flow

The default behavior.

3. Sequential Execution

By default, a Python script executes strictly from top to bottom. Once a line is executed, the interpreter moves to the very next line and never looks back.

# Line 1
name = "Alice"
# Line 2
age = 20
# Line 3
print(name)
# Line 4
print(age)

The Path

The interpreter will ALWAYS execute Line 1. Then Line 2. Then Line 3. Then Line 4. None of it will ever be skipped, and none of it will execute twice.

4. Sequential Example: Recipe Execution

Think of sequential flow like a recipe. You cannot bake the cake before you crack the eggs.

# 1. Gather Ingredients
flour = 2
eggs = 3

# 2. Mix them mathematically
batter = flour + eggs

# 3. Bake (Output)
print(f"Cake ready with {batter} units of batter.")

Conditional Flow

if, elif, and else.

5. Conditional Execution

Conditional flow allows your program to make decisions. It evaluates a Boolean Expression (True or False), and executes a specific block of code ONLY if the condition evaluates to True. Otherwise, it skips it entirely.

6. The Simple 'if' Statement

Structure

The if statement checks a condition. If it is True, the indented block underneath runs.

Syntax:
if <expression>:
    statement(s)
score = 85

if score >= 80:
    print("You passed the test!")
    print("Great job.")

# This line is unindented. It runs sequentially always.
print("Program Ended.")

7. The 'if - else' Statement

The Fork in the Road

The else keyword defines the block of code that should run if the initial if condition was FALSE. It acts as a safety catch-all.

Note: else never takes a condition beside it.

weather = "Rain"

if weather == "Sunny":
    print("Go for a run!")
else:
    print("Stay inside and code.")

# Output: Stay inside and code.

8. The 'elif' Ladder

Short for "else if". When you have multiple specific conditions to check one after the other, you create an elif ladder. Python will check them top-to-bottom, and execute ONLY the first one that returns True.

traffic_light = "Yellow"

if traffic_light == "Red":
    print("Stop!")
elif traffic_light == "Yellow":
    print("Slow Down!")
elif traffic_light == "Green":
    print("Go!")
else:
    print("Malfunction. Proceed with caution.")

9. Conditional Example: Grading System

Converting a numeric score into a Letter Grade using chained limits.

grade = int(input("Enter Marks: "))

if grade >= 90:
    letter = "A"
elif grade >= 80:
    letter = "B"
elif grade >= 70:
    letter = "C"
else:
    letter = "F"

print(f"Your grade is {letter}")

10. Conditional Example: Logic Gates

Using logical and / or inside the conditional evaluation.

user_age = 22
has_ticket = True
is_vip = False

if is_vip or (user_age >= 18 and has_ticket):
    print("Welcome to the concert!")
else:
    print("Security! Escort them out.")

11. Nested Conditionals

If inside an If

You can push indentation multiple levels deep. An inner if statement will only be evaluated if the outer if statement was already proven True.

num = 15

if num > 0:
    print("Positive Number")
    # Nested Block Begins (8 spaces)
    if num % 2 == 0:
        print("And it is Even!")
    else:
        print("And it is Odd!")
else:
    print("Negative Number")

Iterative Flow

The power of Loops.

12. Intro to Iteration

Iterative Flow allows a block of code to be executed over and over again, incredibly fast, until a certain condition is met or all items in a sequence are processed.

Python has two primary loops: for and while.

13. The 'for' Loop

Definite Iteration.

Iterating over Sequences

A for loop is used to iterate over a know sequence (like a String, a List, or a generated Range). It repeats exactly once for every item in that sequence.

word = "PYTHON"

# 'char' becomes a new letter each repetition
for char in word:
    print(char)

# Prints P, Y, T, H, O, N vertically.

14. Generating numeric loops with range()

The range(start, stop, step) function is the backbone of most math-based for-loops in Python. It generates a sequence of numbers up to, but never including, the stop value.

# range(5) generates: 0, 1, 2, 3, 4
for i in range(5):
    print(i)

print("---")

# range(10, 15) generates: 10, 11, 12, 13, 14
for i in range(10, 15):
    print(i)

15. 'for' Loop Example: Multiplication Table

Printing the 5-times table using a range.

num = 5

# Goes from 1 up to 10 (stops BEFORE 11)
for i in range(1, 11):
    ans = num * i
    print(f"{num} x {i} = {ans}")

16. The 'while' Loop

Indefinite Iteration.

Condition-Driven

A while loop doesn't have a fixed sequence length. It simply evaluates a condition. As long as the condition evaluates to True, the block executes.

DANGER: If you never alter the variable inside the block, the condition will remain True forever, causing an Infinite Loop that crashes the machine.

count = 0

# Will loop as long as count is less than 3
while count < 3:
    print(f"Count is {count}")
    # CRITICAL: We must update the state!
    count += 1

print("Loop Finished!")

17. 'while' Loop Example: Input Validation

A classic use case for a while loop is holding a user hostage until they provide a valid password or input.

password = ""

# The loop keeps asking as long as the password is incorrect
while password != "secret123":
    password = input("Enter password to unlock: ")
    if password != "secret123":
        print("Access Denied. Try again.\n")

print("SYSTEM UNLOCKED!")

18. Transfer Statements: break & continue

break

Immediately kills the loop entirely, regardless of the condition, and drops execution out to the next outer line of code.

for i in range(10):
    if i == 3:
        break
    print(i)
# Output: 0, 1, 2 (Then halts forever)

continue

Stops the current iteration early, skips the rest of the block, and jumps straight to the next cycle of the loop.

for i in range(4):
    if i == 2:
        continue
    print(i)
# Output: 0, 1, 3 (Simply skipped 2)
