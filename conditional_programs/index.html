1. What is a Conditional Statement?

A Conditional Statement allows a program to evaluate a Boolean expression and execute different blocks of code depending on whether the result is True or False.

Without conditions, every program would be a boring straight line that does the exact same thing each time it runs.

if

Single condition check.

if-else

Two-way decision fork.

if-elif-else

Multi-way decision ladder.

The Simple 'if'

Execute a block only when the condition is True.

2. Syntax & Flowchart: Simple if

Syntax

if <condition>:
    statement_1
    statement_2

# code here runs always

If the condition is False, the indented block is entirely skipped and execution continues below.

3. Simple if — Example: Checking Age

age = int(input("Enter your age: "))

if age >= 18:
    print("You are eligible to vote!")

print("Thank you for visiting.")

The 'if-else' Fork

Two paths. Always one is guaranteed.

4. Syntax & Flowchart: if-else

Syntax

if <condition>:
    block_A
else:
    block_B

Exactly ONE of the two blocks will always execute. There is no scenario where neither nor both execute.

Flowchart

START

↓
condition?

True ↓ | False ↓
Block A
Block B

↓
END

5. if-else — Example: Even or Odd

num = int(input("Enter a number: "))

if num % 2 == 0:
    print(f"{num} is Even")
else:
    print(f"{num} is Odd")

Logic Trace

The modulus operator % returns the remainder. If the remainder is 0 when divided by 2, the number is perfectly even. Otherwise, it must be odd.

The 'if-elif-else' Ladder

Multi-way branching.

6. Syntax & Flowchart: if-elif-else

Syntax

if <cond_1>:
    block_A
elif <cond_2>:
    block_B
elif <cond_3>:
    block_C
else:
    block_D

Flowchart

cond_1?
→ True → Block A
False ↓
cond_2?
→ True → Block B
False ↓
cond_3?
→ True → Block C
False ↓
else: Block D

Only the FIRST True branch executes.

7. if-elif-else — Example: BMI Classifier

weight = float(input("Weight in Kg: "))
height = float(input("Height in m: "))

bmi = weight / (height ** 2)

if bmi < 18.5:
    print("Underweight")
elif bmi < 25:
    print("Normal Weight")
elif bmi < 30:
    print("Overweight")
else:
    print("Obese")

8. Nested if — Example: Login System

An outer condition checks the username; only if it passes, the inner condition verifies the password.

user = input("Username: ")
pwd = input("Password: ")

if user == "admin":
    if pwd == "pass123":
        print("Login Successful!")
    else:
        print("Wrong Password!")
else:
    print("Unknown User!")

9. Program 1: Absolute Value

Problem Statement

Write a program that takes a number from the user and prints its absolute (positive) value without using the built-in abs() function.

Logic: If the number is negative, multiply by −1. Otherwise, it is already positive.

10. Absolute Value — Flowchart

↓

END

11. Absolute Value — Code

num = float(input("Enter a number: "))

if num < 0:
    num = num * -1 # Flips the sign

print(f"Absolute value = {num}")

Run 1: Input = −7.5

Absolute value = 7.5

Run 2: Input = 12

Absolute value = 12.0

(Already positive, if-block skipped.)

12. Program 2: Sort 3 Numbers (Ascending)

Problem Statement

Accept three numbers from the user and display them in ascending (smallest to largest) order using only conditional statements.

Logic: Compare all possible orderings using nested if-elif-else to find which number is smallest, middle, and largest.

↓ No

→ print b, a, c
↓ ... (continue for all 6 permutations)

14. Sort 3 Numbers — Code

a = int(input("Enter number 1: "))
b = int(input("Enter number 2: "))
c = int(input("Enter number 3: "))

if a <= b <=c:
    print(a, b, c)
elif a <= c <=b:
    print(a, c, b)
elif b <= a <=c:
    print(b, a, c)
elif b <= c <=a:
    print(b, c, a)
elif c <= a <=b:
    print(c, a, b)
else:
    print(c, b, a)

16. Program 3: Divisibility Checker

Problem Statement

Write a program to check whether a given number is divisible by another number entered by the user.

Logic: A number a is divisible by b if the remainder of a / b is exactly zero. (a % b == 0)

17. Divisibility — Flowchart

↓

False ↓
Print "Not Divisible"

↓
END

18. Divisibility — Code

num = int(input("Enter the number: "))
divisor = int(input("Check divisibility by: "))

if divisor == 0:
    print("Error! Division by zero is undefined.")
elif num % divisor == 0:
    print(f"{num} IS divisible by {divisor}")
else:
    print(f"{num} is NOT divisible by {divisor}")

Run 1: 45, 9

45 IS divisible by 9

45 % 9 = 0 ✓

Run 2: 17, 5

17 is NOT divisible by 5

17 % 5 = 2 ✗

19. Extended: Divisibility by Both 3 and 5

A classic CBSE variant: Check if a number is divisible by 3, by 5, by both, or by neither.

n = int(input("Enter a number: "))

if n % 3 == 0 and n % 5 == 0:
    print(f"{n} is divisible by BOTH 3 and 5")
elif n % 3 == 0:
    print(f"{n} is divisible by 3 only")
elif n % 5 == 0:
    print(f"{n} is divisible by 5 only")
else:
    print(f"{n} is divisible by neither 3 nor 5")

Key: The and check MUST come first in the ladder! If we checked n % 3 == 0 alone first, a number like 15 would match that branch and never reach the "both" check.

20. Summary Table

Construct Branches Best For
if 1 (True only) A single guard check. False means skip.
if-else 2 (True / False) Binary decisions. One of two blocks is guaranteed.
if-elif-else Many Multi-way branching. First True branch wins.
Nested if Unlimited depth When an inner condition depends on the outer being True.
