Introduction to Boolean Logic

  • What is it? Algebra used for creating True/False (1/0) statements.
  • Why? The foundation of digital electronics and CPU decision-making.
  • Core values: True (1) or False (0).
  • Basic Operations: NOT, AND, OR.

"Every decision a computer makes is a series of Boolean logic operations."

De Morgan's Laws

Simplifying logic circuits: Equivalence between AND/OR with inversions.

Law 1

¬(A · B) = ¬A + ¬B

Law 2

¬(A + B) = ¬A · ¬B

Why Hex? One Hex digit = exactly 4 binary bits (e.g., F = 1111). It's a human-friendly shorthand for Binary.

Conversion: Other Bases → Decimal

Multiply each digit by its place value and sum them.

Value = Σ (Digit × BasePosition)

Binary → Decimal

1101₂

= (1×2³) + (1×2²) + (0×2¹) + (1×2⁰)

= 8 + 4 + 0 + 1 = 13₁₀

Hex → Decimal

2A₁₆

= (2×16¹) + (10×16⁰) (A=10)

= 32 + 10 = 42₁₀

Conversion: Decimal → Binary

Method: Repeated Division by Base (2). Read remainders backwards.

graph TD A["25 / 2 = 12, Remainder 1"] --> B["12 / 2 = 6, Remainder 0"] B --> C["6 / 2 = 3, Remainder 0"] C --> D["3 / 2 = 1, Remainder 1"] D --> E["1 / 2 = 0, Remainder 1"] F[Read remainders from bottom to top] --> G["Answer: 11001₂"] style F fill:#f9f,stroke:#333,stroke-width:2px

Check: 16+8+0+0+1 = 25₁₀

Encoding Schemes

Computers store everything as binary numbers. Encoding maps characters to specific binary values.

  • Why? So systems can communicate text consistently.
  • How it works: When you type "Hi", the keyboard sends a number for 'H' and a number for 'i'. The OS renders the corresponding glyphs.

'H' → 0x48 (Hex) → 0100 1000 (Binary) → Display "H"

ASCII

American Standard Code for Information Interchange

  • Bits: 7-bits (stored in 1 byte). Capacity: 128 characters.
  • Covers: English letters, digits, punctuation, control characters.

⚠️ Limitation: Only English. No support for Hindi, Arabic, Chinese, etc.

ISCII

Indian Script Code for Information Interchange

  • Bits: 8-bits (1 byte). Capacity: 256 characters.
  • Supports: Devanagari, Bengali, Gujarati, Kannada, Malayalam, etc.
  • Mechanism: Uses a base script structure; font rendering changes the shape.

📌 Legacy: ISCII is now largely obsolete, replaced by the universal standard: Unicode.

Unicode: The Universal Solution

The Problem: ASCII & ISCII couldn't handle the world's languages simultaneously.

The Solution: Unicode provides a unique number for every character, across every language.

  • Capacity: Over 1.1 million code points (U+0000 to U+10FFFF).
  • Coverage: Latin, Devanagari (अ), CJK, Emojis (😊), etc.

'A' = U+0041   |   'अ' (Devanagari) = U+0905

UTF-8, UTF-16, UTF-32

Unicode defines the code point; UTF defines how to store it in memory.

FormatBits per CharDescriptionPros / Cons
UTF-32 32 (4 bytes) Fixed length. ✅ Fast processing. ❌ Wastes bandwidth/space.
UTF-16 16 or 32 bits Variable. 2 bytes for common chars. ✅ Windows/Java standard. ❌ Can still use 4 bytes.
UTF-8 8 to 32 bits Variable. ASCII uses 1 byte. ✅ ✅ Best for the web. Backward compatible with ASCII.

Summary

🔹 Boolean Logic

The language of "Yes/No" decisions. Implemented via Logic Gates. De Morgan's Laws simplify circuits.

🔹 Number Systems

Binary is native to hardware. Octal/Hex are human-readable shortcuts. Conversions rely on place values or grouping.

🔹 Encoding Schemes

ASCII (English) → ISCII (Indian) → Unicode (Universal). UTF-8 is the most efficient and widely used storage format.
