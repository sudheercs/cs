Python Modules

Accessing the Power of Python's Standard Library

Module Importing

A module is a file containing Python definitions and statements. By importing a module, you gain access to its functions, classes, and variables.

1. Using the `import` Statement

Imports the entire module. You must use the module name as a prefix to access its members.

# Syntax: import module_name
# Usage: module_name.function_name()

import math
print(math.pi) # Output: 3.14159...

2. Using the `from ... import` Statement

Imports specific members (functions, variables) directly into the current namespace. You do not need the module name prefix.

# Syntax: from module_name import member1, member2
# Usage: function_name()

from math import sqrt, pi
print(pi) # Output: 3.14159...
print(sqrt(16)) # Output: 4.0

The `math` Module

Provides access to common mathematical functions and constants.

Constants: `pi`, `e`

Mathematical constants (Pi and Euler's number).

print(math.pi)
print(math.e)
`sqrt(x)`

Returns the square root of x.

print(math.sqrt(25)) # 5.0
`ceil(x)` / `floor(x)`

Rounds up (ceiling) or down (floor) to the nearest integer.

print(math.ceil(4.3)) # 5
print(math.floor(4.9)) # 4
`pow(x, y)`

Returns x raised to the power of y ($x^y$).

print(math.pow(2, 3)) # 8.0
`fabs(x)`

Returns the absolute value of x (as a float).

print(math.fabs(-10)) # 10.0
Trigonometric: `sin()`, `cos()`, `tan()`

Returns the sine, cosine, or tangent of an angle given in radians.

print(math.sin(math.pi / 2)) # 1.0

The `random` Module

Generates pseudo-random numbers for various distributions.

`random()`

Returns a random float in the range [0.0, 1.0).

import random
print(random.random())
`randint(a, b)`

Returns a random integer N such that $a \le N \le b$ (inclusive).

print(random.randint(1, 10))
`randrange(start, stop, step)`

Returns a randomly selected element from `range(start, stop, step)`.

print(random.randrange(0, 100, 5))

The `statistics` Module

Provides functions for calculating mathematical statistics of numeric data.

data = [10, 20, 30, 40, 50, 60]
data_b = [10, 20, 30, 30, 40]
import statistics
`mean(data)`

Returns the average (arithmetic mean) of the data.

print(statistics.mean(data)) # 35.0
`median(data)`

Returns the middle value in a sorted data set.

print(statistics.median(data)) # 35.0
`mode(data)`

Returns the most frequent data point (single mode).

print(statistics.mode(data_b)) # 30
