1. Introduction to Boolean Logic

The foundation of modern computing.

🧠

What is it?

Developed by George Boole in the 1800s, Boolean Algebra is a branch of mathematics where the variables have only two possible values: True (1) or False (0).

💻

Application in Computers

Because computers process data using electrical signals (voltage High = 1, Low = 0), Boolean logic maps perfectly to computer hardware. Every computation inside a CPU is just billions of Boolean operations.

🏗️

Logic Gates

A Logic Gate is a physical electronic device (built using transistors) implementing a Boolean function. It takes one or more binary inputs and produces a single binary output.

4. Exclusive OR (XOR) Gate

The "inequality" evaluator.

XOR Gate ( ⊕ )

The XOR (Exclusive OR) gate outputs 1 if and only if the inputs are DIFFERENT.

If both inputs are the same (0,0 or 1,1), it outputs 0. It is heavily used in binary addition and cryptography.

Y = A ⊕ B = A'B + AB'

XOR Truth Table

💡 Real-world Example

A staircase light controlled by two switches (one at bottom, one at top). If you flip either switch (inputs differ), the light toggles state.

5. De Morgan's Laws

Crucial theorems for simplifying Boolean expressions and circuits.

1️⃣

First Law: Break the line, change the sign

The complement of a sum is equal to the product of the complements.

(A + B)' = A' • B'

Circuit Meaning: A NOR gate is logically equivalent to an AND gate with inverted inputs (Bubbled AND).

2️⃣

Second Law: Break the line, change the sign

The complement of a product is equal to the sum of the complements.

(A • B)' = A' + B'

Circuit Meaning: A NAND gate is logically equivalent to an OR gate with inverted inputs (Bubbled OR).

Proving First Law via Truth Table: (A + B)' = A' • B'

6. Logic Circuits in Action

Combining simple gates to do complex arithmetic: The Half Adder.

The Half Adder Circuit

Adds two single binary digits (A and B). Outputs a Sum and a Carry.

How it works:

Sum: Generated by the XOR gate. 1+0=1, 0+1=1. But 1+1=0 (in binary, 1+1 amounts to 10, so sum is 0, carry is 1).

Carry: Generated by the AND gate. Only outputs 1 when both A and B are 1.

📝 Summary

⚙️ Basic Gates

  • AND (•): 1 if ALL inputs are 1.
  • OR (+): 1 if ANY input is 1.
  • NOT ('): Inverts the input.

🌐 Universal & XOR Gates

  • NAND: Inverted AND. Universal.
  • NOR: Inverted OR. Universal.
  • XOR (⊕): 1 if inputs are DIFFERENT.

⚖️ De Morgan's Laws

  • (A + B)' = A' • B'
  • (A • B)' = A' + B'
  • "Break the line, change the sign."

🔌 Logic Circuits

Gates combine to form useful combinational circuits. Example: Half Adder uses XOR for Sum and AND for Carry.
