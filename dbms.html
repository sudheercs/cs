An Introduction to Database Concepts

Understanding the Need for Databases

Presented by: Sudheer Singh

Date: 01/11/2025

What is Data?

Data refers to raw, unorganized facts. By itself it often lacks context or meaning.

Examples

  • Amit (a name)
  • 18 (a number)
  • New Delhi (a place)
  • 95.4 (a score)

Example: “Amit, aged 18, from New Delhi, scored 95.4%.”

What is a Database?

A database is a structured collection of related data organized for efficient access and management.

  • Tables represent entities.
  • Rows (tuples) store records.
  • Columns (attributes) store data types.

DBMS Overview

DBMS = Database Management System. A software that manages data storage, access, and security.

Examples:

  • MySQL, PostgreSQL, Oracle, SQL Server (RDBMS)
  • MongoDB, Cassandra, Firebase (NoSQL)

Relational Model & SQL

Data is represented as tables (relations) with rows and columns.

CREATE TABLE Students (
  RollNo INT PRIMARY KEY,
  Name VARCHAR(50),
  Age INT,
  Course VARCHAR(30)
);

Normalization

  • Reduces redundancy.
  • Improves data integrity.
  • Ensures efficient storage and retrieval.

Summary

  • Databases organize and manage structured data.
  • DBMS ensures integrity, security, and efficiency.
  • SQL is the standard language for relational data.
